# DeepMine

لعبة نجاة وتعدين ثنائية الأبعاد تعمل مباشرة في المتصفح. قاتل موجات كائنات الأعماق، اجمع الموارد، طوّر معداتك، وافتح شخصيات ومناجم جديدة.

## التشغيل

افتح `index.html` مباشرة، أو شغّل خادماً محلياً:

```bash
python -m http.server 8766
```

ثم افتح `http://127.0.0.1:8766`.

## التحكم

- الحركة: `WASD` أو الأسهم
- الاندفاع: `Space`
- الجوال: عصا التحكم وزر الاندفاع على الشاشة

## المميزات

- قتال تلقائي وموجات متصاعدة وزعماء
- 10 شخصيات و6 خرائط و7 أسلحة
- متجر وتطويرات عشوائية وندرات متعددة
- 9 أنواع موارد ونظام إنجازات
- حفظ تلقائي للتقدم والإعدادات
- Object Pooling لدعم أعداد كبيرة من الأعداء
- مؤثرات صوتية وموسيقى إجرائية دون أصول خارجية

جميع الرسومات والواجهة والكود أصلية ومضمنة داخل المشروع.
