# Riftbound Survivors

Survival roguelike عربية أصلية تعمل على الكمبيوتر والجوال، مع حسابات وحفظ سحابي عبر Supabase.

## التشغيل

```bash
python -m http.server 8767
```

ثم افتح `http://127.0.0.1:8767`.

## المحتوى

- 20 شخصية بأساليب وقدرات مختلفة
- 20 سلاحاً عبر ثمانية أنواع ضرر
- 150 عنصراً بتأثيرات إيجابية وسلبية
- 50 عدواً بعشرة سلوكيات
- 100 إنجاز
- 6 خرائط و6 أطوار لعب
- شجرة مهارات ومهام يومية ومسار موسم
- تسجيل وحفظ سحابي ومتصدّرون عبر Supabase
- Object Pooling وشبكة تصادم مكانية للأعداد الكبيرة

مخطط قاعدة البيانات موجود في `supabase/schema.sql`.
